## 教程 TUTORIAL
### 配置设置文件 CONFIGURATION
配置文件是一个py文件，里面有一些配置信息，比如端口、是否记录日志等等。  
请在settings.py里面配置好。
### 启动服务器 STARTING THE SERVER
在终端输入：  
`python server/lhat_server.py`  
即可启动服务器。  
> 注意：这里的`server/lhat_server.py`是相对路径，如果放在项目根目录下，则可以直接输入`lhat_server.py`。
